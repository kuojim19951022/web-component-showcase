適用情境

適合用來展示全站公告、活動訊息或系統維護提醒，讓重要資訊以醒目的方式在頁面上持續曝光，不占用太多版面空間。

技術規格摘要

顯示方式 使用 CSS 動畫與位移效果實作平滑橫向滾動，依內容與容器寬度自動補齊重複項目，避免出現空白斷點。
客製化能力 可以調整滾動速度、每條訊息之間的距離，以及上下留白，並支援從左往右或從右往左兩種滾動方向。
響應式行為 在視窗大小或容器寬度變化時，會自動重新計算需要的內容長度與動畫時間，維持連續且自然的滾動節奏。
封裝與相容性 採用 Web Components 標準與 Shadow DOM 封裝樣式與結構，不依賴第三方套件，支援現代主流瀏覽器。
