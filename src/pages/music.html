頁面說明

本頁以音樂播放器 Web Component 為核心展示目標：使用者點擊播放器後，會由元件接手音樂載入與播放流程。播放器呈現、音樂列表載入與分頁按鈕屬於頁面層互動，用於提供更完整的操作情境。

重點摘要

顯示方式 使用 CSS 動畫與位移效果實作平滑橫向滾動，依內容與容器寬度自動補齊重複項目，避免出現空白斷點。
高度客製 可以調整滾動速度、每條訊息之間的距離，以及上下留白，並支援從左往右或從右往左兩種滾動方向。
響應設計 在視窗大小或容器寬度變化時，會自動重新計算需要的內容長度與動畫時間，維持連續且自然的滾動節奏。
封裝相容 採用 Web Components 標準與 Shadow DOM 封裝樣式與結構，不依賴第三方套件，支援現代主流瀏覽器。
