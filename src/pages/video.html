頁面說明

本頁以 YouTube Modal Web Component 為核心展示目標。使用者點擊影片卡片後，會由元件接手彈窗開啟、資料載入與播放器呈現流程。列表載入與分頁按鈕屬於頁面層互動，用於提供更完整的操作情境。

重點摘要

核心能力 點擊卡片後，以 Web Component 彈窗播放 YouTube 影片。
資料來源模式 可從後端 API 即時取得影片資料，也可用本機或靜態檔當資料來源。
功能開關 可控制標題、描述、相關影片顯示與播放器參數。
播放彈性 可選擇不同播放器類型，在首屏效能（lite-youtube）、整合直覺度（iframe）與進階控制彈性（youtube-iframe-api）間取捨。
效能導向 透過可控的相關影片顯示與延後載入策略，降低初始負擔並優化體驗。
